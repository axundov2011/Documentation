Getting Started

A brief guide to working with Pipeline.

Component Demos

To view a list of the available styled components with markup examples, see [Pipeline Components](../components-pipeline.html) and SinkPay Komponentləri.

Setting up Dev Tools

Unzip the theme package to any folder and open a command line at that location.

Pipeline's dev tools require Node, Git, and the Gulp CLI package. Assuming you have Node and Git installed, the following command installs Gulp Command Line Utility globally so the command can be used anywhere in your system.

When you have installed Gulp, run npm install and the required packages for the Pipeline dev tools will be downloaded to the node_modules directory.

Pipeline employs Browsersync to serve pages from the dist directory. Running will compile the theme, copy all required files to the dist directory and will open a browser window to dist/index.html.

While Gulp is running, Files in the pages, scss and js folders are monitored for changes, which will inject updated CSS or cause a refresh in Browsersync.

Hit Ctrl+C or just close the command line window to stop the server.
